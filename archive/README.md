These splits were produced from earlier inconsistent JSONL source files that pre-date the unified v1 schema. Record formats, key ordering, and unicode handling were non-standard across files, making the splits incompatible with the current v1 schema pipeline. They are kept here for provenance and reproducibility of earlier experiments only — do **not** use them for training. Regenerate splits with `prepare_dataset.py` from the current `*_v1.jsonl` sources.
